## 읽기를 도와주는 방법
읽기는 자연스러운 행위가 아니고, 고도의 훈련이 필요한 작업이다. 따라서 읽기를 도와주기 위해서 디자이너는 어떻게 해야하는가?  

1. 너무 작은 폰트 크기는 배제(최소 8pt 이상)
2. 가독성이 떨어지는 글꼴 사용 지양
3. 산만한 배경 사용하지 말 것
4. 중앙정렬은 최후의 수단으로 사용할 것
5. 시각 계층을 사용하여 구성
6. 평이한 언어로 설명할 것
7. 단순한 언어로 서술할 것
